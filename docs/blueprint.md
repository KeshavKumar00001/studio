# **App Name**: Aushadhalaya

## Core Features:

- Product Catalog: Display medicines across four categories: Ayurvedic, Homeopathic, Supplements, and Allopathic.
- Product Filtering: Enable filtering by category, price range, and other relevant attributes.
- Sorting Options: Provide sorting options like price (low to high, high to low), popularity, and relevance.
- Shopping Cart: Allow users to add products to a cart, update quantities, and view subtotal.
- Checkout Process: Simulate a full checkout process including address submission, payment simulation (no real transactions), and order confirmation.
- User Authentication: Allow users to register, login, and manage their profiles using data in Cloud SQL.
- Recommendation Engine: Suggest other medicines to the user based on what is currently in their cart, acting as a tool to discover products and increase the final order value. AI considers medicine type and attributes of cart contents when selecting the recommendations.

## Style Guidelines:

- Primary color: Light desaturated blue (#ADD8E6) for a calming and trustworthy feel.
- Background color: Off-white (#F5F5F5) to ensure a clean and minimal look.
- Accent color: Muted green (#90EE90) for a subtle, natural feel, drawing connections to traditional Indian medicine.
- Body and headline font: 'PT Sans', a humanist sans-serif for a balance of modernity and readability.
- Use clean, simple line icons for categories, cart, and user profile.
- Maintain a clean, grid-based layout for product listings and ensure a seamless checkout process.
- Subtle transition animations when adding items to the cart and during the checkout process.
